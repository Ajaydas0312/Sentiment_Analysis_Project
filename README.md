Sentiment Analysis Project



This is a basic Python project that can check if a review is positive or negative.



How it works



* Uses Python libraries: NLTK and scikit-learn.



* Trains a model on movie review data.



* You can type in your own review, and it will guess the sentiment.



How to run



1.Make sure you have Python installed.



2.Install required libraries:



pip install nltk scikit-learn





3.Run the program:



python sentiment\_analysis.py





Type your review when prompted.



Example

Enter a review: The movie was amazing and exciting!

Predicted Sentiment: pos
